# Smart Elevator Control System - ER Diagram

This project presents the ER diagram design for a smart elevator control platform used in large commercial buildings like malls, corporate offices, airports, and residential towers.

## 📌 Overview

The system is designed to manage multiple buildings, elevators, floor requests, ride allocations, and maintenance tracking in a scalable and structured way.

## 🧩 Entities Included

* Building
* Floor
* Elevator Shaft
* Elevator
* Elevator-Floor Mapping
* Floor Requests
* Ride / Trip Logs
* Elevator Status
* Maintenance Records

## 🧠 Key Features

* One building can have multiple floors and elevators
* Elevators are assigned to shafts within buildings
* Elevators can serve multiple floors (many-to-many relationship)
* Floor requests are generated by users and assigned to elevators
* Each request results in a ride handled by an elevator
* Ride logs are stored for performance and analytics
* Elevator status (idle, moving, maintenance) is tracked separately
* Maintenance history is preserved without overwriting past data

## 🎯 Purpose

This project is created as part of a database design assignment to model a real-world elevator management system used in high-rise infrastructure.
